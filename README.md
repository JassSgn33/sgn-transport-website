# SGN Transport — Clean, Vercel‑Ready

**How to run locally**
```bash
npm i
npm run dev
```

**How to deploy to Vercel**
1. Create a repo on GitHub and upload everything in this folder.
2. In Vercel → New Project → Import from GitHub → Deploy.
3. Replace `https://formspree.io/f/your-id-here` in `app/page.jsx` with your Formspree IDs.
