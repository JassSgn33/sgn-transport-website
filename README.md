# SGN Transport Inc. — Website

A simple Next.js + Tailwind site for a dump‑truck hauling business.

## Quick Start (Local)

```bash
npm i
npm run dev
```

## Deploy (Vercel)

1. Create a new project at vercel.com and import this repo.
2. Deploy with defaults.
3. Add your domain (e.g., sgntransport.ca) on Vercel → Domains and follow the DNS steps.
4. Update contact info in `app/page.jsx` (`BUSINESS` object).

## Forms (Formspree)

- Replace `https://formspree.io/f/your-id-here` with your actual form IDs (one ID can be reused; hidden `role` field differentiates submissions).
- Or swap in any form provider you prefer.
